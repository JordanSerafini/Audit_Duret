# ANALYSE TRANSVERSALE - SOLUTIONS CRM

**Objectif** : Comparatif synthétique des principales solutions CRM pour PME BTP

---

## SOLUTIONS CRM COMPARÉES

### **ODOO CRM** (Recommandé si Odoo ERP)
- **Prix** : Inclus dans Odoo Enterprise (0€ additionnel)
- **Type** : CRM intégré ERP
- **Forces** : Gratuit si Odoo, intégration parfaite ERP, pipeline visuel
- **Faiblesse** : Moins riche fonctionnellement que HubSpot
- **Adapté Duret** : ✅ Excellent si choix Odoo ERP

### **HUBSPOT CRM**
- **Prix** : Gratuit (base) / 41€/user/mois (Pro) / 150€/user/mois (Enterprise)
- **Type** : CRM marketing & ventes
- **Forces** : Interface moderne, automation marketing, très complet
- **Faiblesses** : Prix élevé en formule Pro+, pas intégré ERP
- **Adapté Duret** : ✅ Si besoin CRM standalone puissant

### **SALESFORCE**
- **Prix** : 25-300€/user/mois selon formule
- **Type** : Leader mondial CRM cloud
- **Forces** : Très complet, écosystème riche
- **Faiblesses** : Complexe, cher, formation longue
- **Adapté Duret** : ❌ Surdimensionné pour PME

### **PIPEDRIVE**
- **Prix** : 14-99€/user/mois
- **Type** : CRM simple axé pipeline commercial
- **Forces** : Simple, visuel, bon rapport qualité-prix
- **Faiblesses** : Moins de fonctionnalités que HubSpot
- **Adapté Duret** : ⚠️ Alternative budget maîtrisé

### **ZOHO CRM**
- **Prix** : 14-52€/user/mois
- **Type** : CRM complet low-cost
- **Forces** : Bon rapport qualité-prix, complet
- **Faiblesses** : Interface moins moderne
- **Adapté Duret** : ⚠️ Alternative crédible

## RECOMMANDATION

### **Scénario 1 : Choix Odoo ERP** → **Odoo CRM**
- Inclus gratuit dans licence
- Intégration native parfaite
- Données centralisées
- **Coût** : 0€ additionnel

### **Scénario 2 : Autre ERP (EBP)** → **HubSpot CRM**
- Version gratuite pour démarrer
- Upgrade Pro si besoin automation (41€/user)
- Meilleur CRM standalone marché
- **Coût** : 0€ (gratuit) ou 4 920€/an (Pro 10 users)
